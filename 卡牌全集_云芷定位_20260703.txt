一、游戏基础机制
━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─━─

【资源】
  内力 (Energy): 每回合回复3点，可存2点到下回合（上限5）
  修为 (Cultivation): 满值突破境界，内力上限+1
  金币 (Gold): 商店消费

【境界】（9阶）
  淬体(0) → 先天 → 筑基 → 金丹 → 元婴 → 化神 → 合体 → 大乘 → 武帝(8)

【门派机制】
  少林 → 禅意(chan)：叠层 → 消耗加伤害/格挡
  武当 → 剑意(jianyi)：叠层 → 消耗加伤害/抽牌
  逍遥 → 无门派资源条 → 靠手牌管理和条件触发

【卡牌类型】
  ATTACK(0) 攻击牌
  SKILL(1)  技能牌
  POWER(2)  能力牌（消耗，持续生效）
  INNER(3)  内功牌（内力相关）
  MOVEMENT(4) 身法牌（抽牌）

【境界加成】
  每升1境界 → 所有伤害/格挡牌 +1

三、云芷（逍遥派）设计文档

━━━━━━━━━━━━━━━━━━━━━━━━━━━
3.1 角色定位
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【一句话定位】
  飘逸灵动 · 条件触发 · 借力打力

【和其他门派的区别】
  少林：站桩叠甲，禅意→爆发（刚猛）
  武当：剑意流转，消耗→强化（均衡）
  逍遥：无固定资源条，靠手牌/牌库/弃牌堆状态触发效果（灵动）

【云芷的核心理念】
  逍遥派没有"禅意"或"剑意"这种专属计数器，
  取而代之的是——通过不同的战场条件触发额外效果。

━━━━━━━━━━━━━━━━━━━━━━━━━━━
3.2 四大流派方向
━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─────────────────────────────────────────────────────────────────┐
│ 流派A：飘逸流（手牌数量联动）                                    │
│   "手牌越多，伤害越高"                                           │
│   代表效果：手牌数×2伤害、手牌≥5时额外效果、弃牌强化               │
│   核心思路：少出牌、攒手牌、一波带走                               │
│   已有：天ft 折梅手（手牌≤3时+5伤害）                            │
├─────────────────────────────────────────────────────────────────┤
│ 流派B：借力流（利用敌人状态）                                    │
│   "敌人的就是我的"                                              │
│   代表效果：根据敌人护盾/HP/意图增强                              │
│   核心思路：观察敌人，选择最佳时机出牌                             │
│   已有：北冥神掌（吸HP）                                         │
├─────────────────────────────────────────────────────────────────┤
│ 流派C：奇策流（条件增强）                                        │
│   "特定条件下效果翻倍"                                           │
│   代表效果：上一张是攻击则强化、出牌顺序combo、消耗手牌换收益        │
│   核心思路：规划出牌顺序，运营combo                                │
│   已有：凌波微步（下张牌-1费）                                    │
├─────────────────────────────────────────────────────────────────┤
│ 流派D：内息流（内力管理）                                        │
│   "内力不仅是资源，也是武器"                                     │
│   代表效果：额外消耗内力换收益、满内力时加成、存内力过回合           │
│   核心思路：管理内力上限，选择何时留内力何时爆发                    │
│   已有：内力存储系统（最多存2点到下回合）                          │
└─────────────────────────────────────────────────────────────────┘

3.3 卡牌设计目标（30+张）
━━━━━━━━━━━━━━━━━━━━━━━━━━━

  飘逸流（8~10张）: 手牌联动
  借力流（5~7张）: 敌人状态联动  
  奇策流（8~10张）: combo/条件触发
  内息流（4~6张）: 内力管理
  核心能力（2张）: POWER级核心牌

━━━━━━━━━━━━━━━━━━━━━━━━━━━
4.1 回合流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 玩家回合开始
  2. POWER效果触发（达摩一苇/八荒六合/太极两仪重置）
  3. 抽牌（4张/回合）
  4. 内力回复（存2点→上限5）
  5. 出牌阶段（自由操作）
  6. 点击"结束回合"
  7. 敌人回合
  8. 敌人被动触发（精英/Boss额外护盾）
  9. 敌人执行意图（攻击或防御）
  10. 敌人规划下回合意图
  11. 切回玩家回合

━━━━━━━━━━━━━━━━━━━━━━━━━━━
4.2 敌人难度系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━

  普通层: 基础HP×1.0, 攻击3-6
  精英层: HP×1.5, 攻击+20%, 开局自带15%盾, 每回合回4%盾
  Boss层:  HP×2.5, 三段狂暴（HP>66%/33%/濒死）

━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━

  敌人每回合显示意图：攻击 或 防御
  各类型倾向不同：
  普通: 55%攻击
  精英: 40%攻击（更倾向防御）
  Boss: 一阶段30%攻击 → 二阶段50% → 三阶段100%狂暴
